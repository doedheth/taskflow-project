# Project Overview - TaskFlow

**Project Name:** TaskFlow (projectSAP)
**Version:** 1.0.0
**Project Type:** Multi-part Monorepo (React Frontend + Express Backend)
**Architecture Style:** Layered Architecture (v2 OOP Refactored)
**Primary Language:** TypeScript

## Executive Summary
TaskFlow adalah sistem manajemen tugas dan pemeliharaan (Task & Maintenance Management System) yang dirancang untuk mengintegrasikan alur kerja operasional dengan pemeliharaan aset. Sistem ini mendukung manajemen tiket (Kanban), penjadwalan pemeliharaan preventif, pelacakan downtime, dan analisis berbasis AI untuk prediksi pemeliharaan.

## Technology Stack Summary

| Part | Role | Technologies |
|------|------|--------------|
| **task-manager-client** | Web Frontend | React 18.3, Vite 5.4, TypeScript, TailwindCSS, TanStack Query, Recharts, dnd-kit |
| **task-manager-server** | Backend API | Node.js, Express 4.21, TypeScript, SQLite (sql.js), OpenAI SDK, node-cron |

## Repository Structure
Proyek ini menggunakan struktur *multi-part* dengan pemisahan yang jelas antara klien dan server:
- `task-manager-client/`: Aplikasi frontend berbasis React.
- `task-manager-server/`: API backend berbasis Node.js/Express.
- `docs/`: Dokumentasi proyek dan laporan pemindaian.
- `_bmad/`: Konfigurasi dan agen BMAD untuk pengembangan berbantuan AI.

## Key Features
- **Manajemen Tiket & Kanban:** CRUD tiket dengan tipe (bug, task, story, epic) dan papan Kanban interaktif.
- **Maintenance Management:** Registri aset, Work Order (WO), dan pemeliharaan preventif (PM).
- **Role-Based Dashboard:** Dashboard khusus untuk Supervisor, Member, Manager, dan Admin.
- **AI Enhancement:** Chatbot sadar konteks, prioritas tugas otomatis, dan analisis pemeliharaan prediktif.
- **Energy Monitoring & Analytics:** Integrasi Huawei FusionSolar dan PLN Induk (Webhook) untuk pemantauan beban energi dan kalkulasi penghematan (Net Revenue) secara real-time.
- **Public Slideshow System:** Tampilan informasi agregat (KPI, Produksi, OEE, Solar) yang dioptimalkan untuk tampilan TV publik/kantor.
- **Downtime & KPI Tracking:** Pelacakan waktu henti mesin dan visualisasi indikator kinerja utama (KPI).
